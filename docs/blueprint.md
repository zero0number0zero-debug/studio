# **App Name**: ProjectPM

## Core Features:

- Login: Secure user login using email/password or social authentication.
- Registration: User registration with email verification and password security.
- Forgot Password: Password reset functionality with email verification.
- Project Management (PMBOK): Manage projects according to PMBOK guidelines, including initiation, planning, execution, monitoring, controlling, and closure.
- Code Storage: Integration with GitHub for storing and managing project code.
- Data Storage: Utilize Firestore for storing project data, user information, and other relevant details.
- API Security: Secure API endpoints with authentication and authorization mechanisms to protect sensitive data.

## Style Guidelines:

- Primary color: Navy blue (#2E3A87) for a professional and trustworthy feel.
- Secondary color: Light gray (#F5F5F5) for background and content areas.
- Accent color: Teal (#008080) to highlight interactive elements and CTAs.
- Body font: 'Roboto' for readability and a modern aesthetic.
- Header font: 'Montserrat' for a bold and clear presentation.
- Use a consistent set of icons from a library like Font Awesome to represent project elements.
- Maintain a clean and organized layout with intuitive navigation and clear information hierarchy.
- Subtle animations for transitions and feedback to improve user experience.
